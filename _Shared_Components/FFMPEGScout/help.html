COM Class Name:
FFMpegScout.Application

Properties:

Width (USHORT) Frame width
Height (USHORT) Frame height
InputFileName (BSTR) Name of the input file (required)
OutputFileName (BSTR) Name of the output file (required)
AudioBitrate (USHORT) Target audio bitrate
VideoBitrate (USHORT) Target video bitrate
VideoType (BYTE)

Target video type:
0=none, 1=vcd, 2=svcd, 3=dvd, 4=dv, 5=pal-vcd, 6=ntsc-svcd
RemoveAudioStream (BOOL) Audio stream will be removed from output if set to TRUE
RemoveVideoStream (BOOL) Video stream will be removed from output if set to TRUE
ForcedCodecName (BSTR) Forced codec name for video (such as "mpeg4", "mjpeg" etc)
ForcedAudioCodecName (BSTR) Forced audio codec name (such as "mp2" etc)
PathToFFMpeg (BSTR) Fully qualified path name for the "ffmpeg.exe" (by default it will look for ffmpeg.exe at the current directory)

Methods:

Execute Starts conversion
