Using the .NET API in your Project

The DB XML .NET API assembly (bin/dbxml_dotnet.dll) has been installed into the global assembly cache (GAC) for you. You should be able to access it through the "Add Reference" dialog in visual studio, under the ".NET" tab, called "Berkeley DB XML .NET API". The "Add Reference" dialog box can be found by right clicking on your project in Visual Studio, and selecting "Add Reference".

Berkeley DB XML is written in C++, and relies on a number of native DLLs. You will need to make sure that these DLLs are on the path when you run your program. If you have selected the default install directory for both DB XML and the .NET API, then the DLLs you will need are:

DLLLocationPurpose

libdbxml21.dllC:\Program Files\Sleepycat Software\Berkeley DB XML 2.1.8\binDB XML C++ DLL
libdb43.dllC:\Program Files\Sleepycat Software\Berkeley DB XML 2.1.8\binBerkeley DB C++ DLL
libxquery11.dllC:\Program Files\Sleepycat Software\Berkeley DB XML 2.1.8\binBerkeley XQuery DLL
Pathan_7.1.dllC:\Program Files\Sleepycat Software\Berkeley DB XML 2.1.8\binPathan XPath2 DLL
xerces-c_2_6.dllC:\Program Files\Sleepycat Software\Berkeley DB XML 2.1.8\binApache Xerces-C DLL

These instructions use a Berkeley DB XML release number of 2.1.8. If that is not your release, adjust your paths accordingly.

API Documentation

There is HTML API documentation available here, and a windows help file available here.

Context sensitive help has already been installed into Visual Studio .NET, if you have it installed. This can be accessed by pressing F1 on any DB XML .NET term in the Visual Studio editor.

Getting Started Examples

There are a number of example programs provided in the examples/src directory written in C#, if you chose to install them. These have been translated from the Java getting started guide's examples. Project files for the examples have also been included and can be accessed via the examples solution file (examples/examples.sln). A link to the examples solution file should have been installed in your start menu. The examples are very simple programs, that exhibit a wide range of common DB XML use cases using the .NET API.

Unfortunately, there is currently no .NET version of the getting started guide, but the Java version can be found here.
