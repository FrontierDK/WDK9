If the operation is to be transaction-protected, the txn parameter is an Transaction handle, otherwise it is a null reference (Nothing in Visual Basic).

The name of the document to be inserted.
contents
The content of the document to be inserted.

The update context to use for the operation.
config
Configuration settings for this operation

Return Value

The name of the document that has been inserted.

Remarks

Note that the name used for the document must be unique in the container or an exception is thrown. The configuration setting GenerateName can be used to generate a name. To change a document that already exists in the container, use UpdateDocument.

The document content is indexed according to the container indexing specification. The indexer supports the Xerces content encodings and expects the content to be well-formed, but it need not be valid.
