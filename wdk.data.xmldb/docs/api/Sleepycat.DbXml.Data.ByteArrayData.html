Remarks

This property will copy the memory region pointed to by this Data object into a new byte array, if it is read.

If written, the byte array will be pinned in memory using Alloc, and then an IntPtr that points to it's data is obtained.

This will not copy the byte array, so changes made to it whilst the Data object is still active may result in undetermined effects.
