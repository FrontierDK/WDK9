Resolver.ResolveEntity Method 

When implemented, should resolve a System ID and Public ID to a new InputStream. If the IDs cannot be resolved by this resolver, this method should return a null reference (Nothing in Visual Basic). The InputStream object will be disposed of by the caller.

Return Value

The InputStream that the system and public ID resolve to, or a null reference (Nothing in Visual Basic).
