Remarks

The type of the value may be any one of the Value.ValueType constants. There are convenience operators for specially handling the types of STRING, DOUBLE, BOOLEAN, DATE_TIME, and NODE.

The Value class provides several constructors, each of which maps a Java type or Berkeley DB XML class onto an appropriate Value type.

This class implements the IDisposable interface, and the Dispose method must always be called on it to allow the clean up of unmanaged memory and container handles.

Note    This class is analogue to the C++ class XmlValue.
