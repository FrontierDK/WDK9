Parameters

kbyte
A checkpoint will be performed if more than kbytes of log data have been written since the last checkpoint. Set to 0 to disable this parameter.
min
A checkpoint is performed if more than min minutes have passed since the last checkpoint. Set to 0 to disable this parameter.
force
If set to true, force a checkpoint, even if there has been no activity since the last checkpoint.
