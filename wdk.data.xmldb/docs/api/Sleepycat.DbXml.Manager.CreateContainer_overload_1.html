Parameters

txn
The transaction handle to use for this container creation, or a null reference (Nothing in Visual Basic).
name

The container's name. The container is created relative to the underlying environment's home directory (see Manager for more information) unless an absolute path is used for the name; in that case the container is created in the location identified by the path.

The name provided here must be unique for the environment or an exception is thrown.

Remarks

The container is created using the configuration from the DefaultContainerConfig property.
