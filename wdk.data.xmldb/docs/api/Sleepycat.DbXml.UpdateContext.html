Remarks

An UpdateContext can be created by calling Manager.CreateUpdateContext.

Passing an object of this class through the Container interface methods provides performance benefits, as the context contains objects that can be reused multiple times.

This class implements the IDisposable interface, and the Dispose method should be called on it to avoid holding unnecessary unmanaged memory.

Note    This class is analogue to the C++ class XmlUpdateContext.
