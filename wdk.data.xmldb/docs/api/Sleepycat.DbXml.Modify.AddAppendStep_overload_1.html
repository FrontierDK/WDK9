Parameters

selectionExpr
The XQuery expression used to target the location in the document where the modification is to be performed. Use Manager.Prepare to create the QueryExpression. The query that you provide must target zero or more nodes, or an exception is thrown. If zero nodes are selected, no modifications are performed when Execute is called.
type
The type of information to be inserted. The value provided here determines whether the name or content parameter is required.
name
The name of the node, attribute, or processing instruction to insert. This parameter is ignored if type is Text or Comment.
content
The content to insert. If the type specified is Element, this content is parsed for XML markup it contains.

Remarks

If the operation's target is an attribute node an exception is thrown at modification execution time.

If the content to be added is an attribute, the content is added to the targeted node's parent node. For any other type of content, the content is inserted into the document immediately before the targeted node's start tag.

Documents resulting from this modification must be well-formed XML or an exception is thrown at modification execution time, if the documents affected are written back to their respective containers.
