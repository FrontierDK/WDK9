Remarks

Container exposes methods for managing (putting and deleting) Document objects, managing indices, and retrieving container statistics.

If the container has never before been opened, use CreateContainer to instantiate an Container object. If the container already exists, use OpenContainer instead. Containers are always opened until the last referencing handle is destroyed.

You can delete containers using RemoveContainer and rename containers using RenameContainer.

This class implements the IDisposable interface, and the Dispose method must always be called on it to allow the clean up of unmanaged memory and file handles, and the flushing of any outstanding database data.

Note    This class is analogue to the C++ class XmlContainer.
