Remarks

Upon completing the modification, the modified document is automatically updated in the backing Container for you, unless UpdateContext.ApplyChangesToContainer has been set to false.

Note that the modification steps are executed in the order that they were specified to this object. Modification steps are specified using AddAppendStep, AddInsertBeforeStep, AddInsertAfterStep, AddRemoveStep, AddRenameStep and AddUpdateStep.

This method can thrown an exception for any number of reasons; the most common being that an illegal modification was requested. In addition, if a resulting document is not well-formed XML, this method will throw a parsing exception if that document is written to its container.
