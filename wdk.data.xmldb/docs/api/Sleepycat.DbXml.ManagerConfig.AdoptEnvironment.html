ManagerConfig.AdoptEnvironment Property

Sets whether the Manager object will close and delete the underlying Environment handle at the end of the Manager's life.

Remarks

By default, the Environment object is not adopted.

If this option is set, then Dispose does not need to be called on the Environment object used to construct the Manager.
