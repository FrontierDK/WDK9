env
The environment handle to use for the underlying database environment. The handle provided here must be opened.

Remarks

A default ManagerConfig object is used in this constructor.

The Berkeley DB subsystems initiated by this environment (for example, transactions, logging, the memory pool), are the subsystems that are available to Berkeley DB XML when operations are performed using this manager object.
