Remarks

You can obtain an instance of this object using one of Manager.CreateInputStream, Manager.CreateStdInInputStream, Manager.CreateLocalFileInputStream, Manager.CreateURLInputStream or Document.StreamContent. You use instances of this class with Container.PutDocument and Document.StreamContent.

You can manually retrieve the contents of the input stream using ReadBytes and CurrentPosition.

InputStream is an abstract class, and can be subclassed, and an instance passed to any of the methods that take it as a parameter. This is especially useful for streaming XML from an application directly into Berkeley DB XML without first converting it to a string.

This class implements the IDisposable interface, and the Dispose method must always be called on it to allow the clean up of unmanaged memory and file handles.

Note    This class is analogue to the C++ class XmlInputStream.
