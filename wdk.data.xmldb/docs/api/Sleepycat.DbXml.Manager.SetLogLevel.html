Remarks

Berkeley DB XML can be configured to generate a stream of messages to help application debugging. The messages are categorized by subsystem, and by importance. The messages are sent to the output stream that is configured in the Berkeley DB environment associated with the Manager generating the message. The output is sent to System.err if no environment is associated with the Manager.
