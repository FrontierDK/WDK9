ResolveCollection When implemented, should resolve the given URI to a Results object containing node Value objects . If the URI cannot be resolved by this resolver, this method should return a null reference (Nothing in Visual Basic).
ResolveDocument When implemented, should resolve the given URI to a Document. If the URI cannot be resolved by this resolver, this method should return a null reference (Nothing in Visual Basic).
ResolveEntity When implemented, should resolve a System ID and Public ID to a new InputStream. If the IDs cannot be resolved by this resolver, this method should return a null reference (Nothing in Visual Basic). The InputStream object will be disposed of by the caller.
ResolveSchema When implemented, should resolve schema location and namespace information to a new InputStream. If this information cannot be resolved by this resolver, this method should return a null reference (Nothing in Visual Basic). The InputStream object will be disposed of by the caller.
