Remarks

File URLs are always supported by this method. URLs that require network access (for example, http://...) are supported only if Xerces was compiled with socket support.

Note that there is no attempt to ensure that the URI referenced contains well-formed or valid XML. Exceptions may be thrown at the time that this input stream is actually read if the stream does not contain well-formed, or valid XML.
