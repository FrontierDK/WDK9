Namespace hierarchy

Classes

Class Description
Container The Container class encapsulates a document container and its related indices and statistics.
ContainerConfig Configuration settings for an Container instance.
Data The Data class encapsulates a Berkeley DB Dbt, which is the fundamental storage object for the underlying DB databases.
DbXmlException A represention of an error condition that has occurred within the Berkeley DB XML system.
Document The unit of storage within an XmlContainer.
DocumentConfig Used to configure operations that operate on Document objects.
IndexSpecification The encapsulation of the indexing specification of a container.
IndexSpecification.Entry An indexing strategy.
InputStream Used to read and write XML data.
Manager Provides a high-level object used to manage various aspects of Berkeley DB XML usage.
ManagerConfig Used to configure a Manager instance via its constructors.
MetaData A class that encapsulates document metadata.
MetaDataEnumerator Provides an enumeration over a Document's metadata.
Modify The context within which a set of one or more documents specified by an XQuery query can be modified in place.
Node The abstract class that represents a Node
QueryContext The QueryContext class encapsulates the context within which a query is performed.
QueryExpression An QueryExpression represents a parsed XQuery expression, and is created by a call to Manager.Prepare.
Resolver Provides an unimplemented base class that is used for file resolution policy.
Results The Results class encapsulates the results of a query.
Statistics The Statistics class encapsulates statistical information about the number of keys in existence for a given index.
Transaction The Transaction class is the handle for a transaction.
TransactionConfig A class that specifies the configuration of a Manager object
UpdateContext The UpdateContext class encapsulates the context within which update operations are performed against a Container.
Value The Value class encapsulates the value of a node in an XML document.
VerifyConfig Specifies the attributes of a verification operation.
