Remarks

A Manager object is used to perform activities such as container management (including creation and open), preparing XQuery queries, executing one-off queries, creating transaction objects, creating update and query context objects, and creating input streams.

This class implements the IDisposable interface, and the Dispose method must always be called on it to allow the clean up of unmanaged memory and file handles, and the flushing of any outstanding database data.

Note    This class is analogue to the C++ class XmlManager.
