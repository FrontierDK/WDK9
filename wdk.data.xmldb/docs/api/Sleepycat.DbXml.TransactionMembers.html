Public Instance Methods

Abort Cause an abnormal termination of the transaction.
Commit End the transaction.
CommitNoSync End the transaction without flushing the log.
CommitSync End the transaction and synchronously flush the log.
CreateChild Creates a child transaction of this transaction. While this child transaction is active (has been neither committed nor aborted), the parent transaction may not issue any operations except for Commit or Abort.
Dispose The Dispose method must always be called to allow the Container to clean up unmanaged memory and file handles, and to flush any outstanding database data.
