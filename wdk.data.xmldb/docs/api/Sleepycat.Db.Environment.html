Thread Safety

This type is safe for multithreaded operations.

If EnvironementConfig.Threaded = true, this instance is thread safe.

Remarks

To open an existing environment with default attributes the application may use a default environment configuration object: 

// Open an environment handle with default attributes.
Environment env = new Environment(home, new EnvironmentConfig());
Note that many Environment objects may access a single environment.

To create an environment or customize attributes, the application should customize the configuration class. For example: 

 EnvironmentConfig envConfig = new EnvironmentConfig();
 envConfig.Transactional = true;
 envConfig.AllowCreate = true;
 envConfig.CacheSize = 1000000;

 Environment newlyCreatedEnv = new Environment(home, envConfig);

An environment handle is an Environment instance. More than one Environment instance may be created for the same physical directory, which is the same as saying that more than one Environment handle may be open at one time for a given environment.

This class implements the IDisposable interface, and the Dispose method must always be called on it to allow the clean up of unmanaged memory and file handles, and the flushing of any outstanding database data.

Note    This class does not currently implement all of the methods provided by Berkeley DB's DbEnv object. Greater support is planned in the future.
