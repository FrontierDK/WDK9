Remarks

This class aims to implement all the features of a read only W3C DOM node. It adds special support for retrieving an attribute's owner element.

This class implements the IDisposable interface, and the Dispose method should be called on it to avoid holding unnecessary unmanaged memory.

This class is also implemented by Document, which adds more functionality to it.
