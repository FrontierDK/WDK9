Parameters

txn
If the operation is to be transaction-protected, the txn parameter is an Transaction handle, otherwise it is a null reference (Nothing in Visual Basic).
context
The update context to use for the operation.
entry
The index entry to use for the look up.
value
Provides the value to which equality indices must be equal. This parameter is required when doing a value lookup, otherwise it can be a null reference (Nothing in Visual Basic).

Return Value

A Results object containing the result of the index lookup.

Remarks

By default, entire documents are returned by this method. However, if the container is of type NodeContainer (the default container type), and if IndexNodes is set for the container, then this method will return the individual nodes referenced by the index keys.

Exceptions

DbXmlException If a failure occurs.
